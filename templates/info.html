How to use?

Although the app was designed to be as much bugproof as possible, there are some instructions that must be mentioned.

Paste lyrics where chords and text are separated in columns - chords on the top or on the bottom of the text, not in the same line.

Also, chords should be written in capital letters and there should be blank space before and after every chord --> Am7/C is well written chord.
However, if you follow this rules and still find an error, you can double-click chord to change it to text or other way around.

About project and developer

This web app is a result of the final middle school project. (Tehnička škola Daruvar, CRO)

This website is Flask an JS based project by Dominik Milde. The idea for this project came from my passion for music and computer programming.
